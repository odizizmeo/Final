หัวข้อวิจัย

A Hardware-Based Biometric Authentication System and Secure Token Encryption

บทคัดย่อ

งานวิจัยนี้วัตถุประสงค์พัฒนาระบบยืนยันตัวตน โดยใช้ฮาร์ดแวร์เป็นส่วนสำคัญหลัก เช่น Raspberry Pi, Arduino, Fingerprint Scan หรืออุปกรณ์ Biometric ต่างๆ ร่วมกับหลักการสร้าง Token การเข้ารหัสต่างๆ ให้การ Authentication เข้าถึงระบบสารสนเทศหรือบริการออนไลน์มีความปลอดภัยสูง กระบวนการยืนยันตัวตนจะประกอบไปด้วยการอ่านลายนิ้วมือ และการสร้าง Token แบบใช้ครั้งเดียว (One-Time-Token) ที่เข้ารหัสด้วย HMAC-SHA512 ก่อนส่งไปยัง Authentication Server ผ่าน HTTPS ป้องกันการปลอมแปลงข้อมูลและ การเข้าถึงโดยไม่ได้รับอนุญาต

วัตถุประสงค์ของการวิจัย

    1. เพื่อสร้างระบบยืนยันตัวตนที่ปลอดภัย: พัฒนาระบบยืนยันตัวตนที่สามารถป้องกันการโจมตีและเข้าถึงโดยไม่ได้รับอนุญาตได้ดียิ่งขึ้นเมื่อเทียบกับวิธีการใช้รหัสผ่านแบบดั้งเดิม

    2. เพื่อเพิ่มความสะดวกในการใช้งาน: ออกแบบฮาร์ดแวร์และซอฟต์แวร์ให้ใช้งานง่ายและรวดเร็วสำหรับผู้ใช้ โดยไม่ต้องจดจำรหัสผ่านที่ซับซ้อน

    3. เพื่อเป็นต้นแบบสำหรับการนำไปประยุกต์ใช้: สร้างโปรเจกต์ที่เป็นตัวอย่างสำหรับการนำแนวคิดนี้ไปประยุกต์ใช้ในระบบสารสนเทศอื่นๆ ที่ต้องการความปลอดภัยสูง เช่น ระบบธนาคาร, ระบบการแพทย์ หรือระบบควบคุมการเข้าถึงในองค์กร

ขอบเขตของการวิจัย

การวิจัยนี้จะครอบคลุม [ขอบเขตการวิจัย] โดยใช้ [เครื่องมือ/เทคโนโลยี] ในการพัฒนาและทดสอบ

ประโยชน์ที่คิดว่าจะได้รับ

    1. เพิ่มความปลอดภัยของข้อมูล ลดความเสี่ยงจากการถูกขโมยรหัสผ่านและการเข้าถึงข้อมูลโดยไม่ได้รับอนุญาต ซึ่งเป็นปัญหาใหญ่ในระบบสารสนเทศปัจจุบัน

    2. ลดภาระของผู้ใช้ ผู้ใช้ไม่ต้องกังวลเรื่องการตั้งรหัสผ่านที่เดาได้ยากหรือความเสี่ยงจากการถูกขโมยรหัสผ่าน

    3. เพิ่มความน่าเชื่อถือของระบบ สร้างความเชื่อมั่นให้กับผู้ใช้งานและองค์กรว่าระบบมีการป้องกันที่แข็งแกร่งและได้มาตรฐาน

    4. เปิดโอกาสในการพัฒนาต่อยอด ผลลัพธ์ของโปรเจกต์นี้สามารถนำไปเป็นพื้นฐานในการพัฒนาโซลูชันด้านความปลอดภัยอื่นๆ ที่ใช้ฮาร์ดแวร์เป็นหลัก เช่น การยืนยันตัวตนแบบหลายปัจจัย (Multi-factor Authentication) ที่ใช้ฮาร์ดแวร์ร่วมกับไบโอเมตริกซ์
